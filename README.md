# アプリケーション名
calendar_app

# アプリケーション概要
ユーザーごとの予定をカレンダーに入力して管理できる。

# テスト用アカウント

# 利用方法
## カレンダーに予定を記入する
1.トップページの「新規登録」ボタンからユーザー新規登録を行う
2.ログイン後トップページの「Calendar」ボタンからカレンダーページへ遷移する
3.「予定の追加」ボタンから予定入力ページへ遷移し、入力する日付を選択し予定を入力する

# アプリケーションを作成した背景
今後就職活動をしていく中で、予定表が必要になり、どうせなら自分で予定表のアプリケーションを作成してみようと思ったため。
